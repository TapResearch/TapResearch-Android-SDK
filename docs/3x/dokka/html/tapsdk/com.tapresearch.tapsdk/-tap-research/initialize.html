initialize

fun initialize(apiToken: String, userIdentifier: String, activity: Activity, rewardCallback: TRRewardCallback? = null, errorCallback: TRErrorCallback, sdkReadyCallback: TRSdkReadyCallback, qqDataCallback: TRQQDataCallback? = null, initOptions: TapInitOptions? = null)

Initialize the TapResearch SDK. Must be called before most TapResearch api.

Parameters

apiToken

  • The API Token; provided by TapResearch team.

userIdentifier

  • The unique user identifier; can be set again by calling setUserIdentifier.

activity

  • Activity; application Context can also be used.

rewardCallback

  • Optional reward callback; can also call SetRewardCallback

errorCallback

  • Mandatory error callback; e.g. initialization errors.

sdkReadyCallback

  • Mandatory SDK ready callback; let's you know the SDK is ready for use.

qqDataCallback

  • Optional Quick Question data callback; also see setQuickQuestionCallback.

initOptions

  • Optional TapInitOptions. Contains optional targeting parameters.
